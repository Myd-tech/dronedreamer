## 🌟 项目介绍

**DroneDreamer** 是一个低空无人机六视角数据集，致力于拓展世界模型的应用边界，提供六视角图像以增强世界模型的全方位感知能力。

---

## 📁 数据集示例

我们在 `assets/` 目录中提供了多个真实采集的六视角视频片段，展示原始输入数据的多样性与复杂性。每个视频由 6 个视角按 **2×3 网格**拼接而成（布局：`[300, 0, 60] / [240, 180, 120]`）。

> 💡 提示：所有视频均可直接下载（右键链接 → “链接另存为”）。数据来自真实无人机平台，在不同天气与光照条件下采集。
